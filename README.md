# Lisp repository

Run main.lisp, i.e.

> (load "main.lisp")


# Key Mapping

| col | description |
|-----|-------------|
| <kbd>w</kbd> | move forward |
| <kbd>s</kbd> | move backward |
| <kbd>d</kbd> | move right |
| <kbd>up</kbd> | look up |
| <kbd>down</kbd> | look down |
| <kbd>left</kbd> | look left |
| <kbd>right</kbd> | look right |
| <kbd>esc</kbd> | quit program |
